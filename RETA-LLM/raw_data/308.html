  • 财政金融学院
    http://sf.ruc.edu.cn/

    中国人民大学财政金融学院是新中国第一个培养财政金融领域高级人才的基地，为新中国财政金融学科的建立和发展做出了开拓性贡献。经过70余年的积淀与建设，学院已经成为国内外知名的财政金融教学科研重镇，被誉为“中国财政金融高等教育的领头雁”。

    本科专业：
    • 保险学
      •
    • 信用管理
      •
    • 金融工程
      •
    • 金融学
      •
    • 税收学
      •
    • 财政学
      •

  • 统计学院
    http://stat.ruc.edu.cn

    中国人民大学统计学科始建于1950年，两年后设立统计学系，2003年成立统计学院。经过几代统计人的奋斗，我校统计学科取得了令人瞩目的成绩，在中国乃至亚洲处于领先地位，成为中国统计教育的一面旗帜。

    本科专业：
    • 统计学
      •
    • 应用统计学
      •
    • 数据科学与大数据技术
      •
    • 经济统计学
      •

  • 商学院
    http://www.rmbs.ruc.edu.cn

    中国人民大学商学院是新中国工商管理教育的奠基者，也是中国管理教育重要的开创者、引领者。学院以“贡献中国管理智慧、培养全球领袖人才”为使命，致力于成为“最懂中国管理的世界一流商学院”。

    本科专业：
    • 管理科学
      •
    • 财务管理
      •
    • 会计学
      •
    • 贸易经济
      •
    • 市场营销
      •
    • 工商管理
      •

  • 劳动人事学院
    http://slhr.ruc.edu.cn/

    中国人民大学劳动人事学院是“中国人力资源管理第一品牌”，中国人力资源管理研究的发源地，人力资源管理学术领域最早的开创者，人力资源管理思想的引领者，中国人力资源管理实践的推动者。

    本科专业：
    • 人力资源管理
      •
    • 劳动与社会保障
      •
    • 劳动关系
      •
    • 劳动经济学
      •

  • 经济学院
    http://econ.ruc.edu.cn

    中国人民大学经济学院于1988年正式组建，是新中国经济学科的重要奠基者和开拓者。学院以马克思主义政治经济学为支撑，重点构建宏观经济理论与政策、微观经济理论与政策、新兴学科三大学科集群，长期以来雄踞国内第一、世界一流的优势地位。

    本科专业：
    • 经济学
      •
    • 国际经济与贸易
      •
    • 数字经济
      •

  • 中法学院
    http://ifc.ruc.edu.cn

    中法学院是由中国人民大学与法国索邦大学、蒙彼利埃保罗—瓦莱里大学、KEDGE商学院合作创办的第一家以人文社会科学为主的中外合作办学机构，是隶属于中国人民大学的非独立法人办学单位。

    本科专业：
    • 法语(中外合作办学)
      •
    • 国民经济管理(中外合作办学)

      •
    • 数学与应用数学（中外合作办学）
      •
    • 人力资源管理（中外合作办学）
      •
    • 传播学（中外合作办学）（国际新闻传播方向）

  • 应用经济学院
    http://ae.ruc.edu.cn/

    2019年，中国人民大学立足改革开放的伟大实践，为加快构建新的理论框架和研究范式，高水平推动“双一流”学科建设，组建应用经济学院。应用经济学科连续三轮在教育部组织的一级学科评估中排名全国第一 ，在第四轮评估中获评A+，2022年再次进入国家“世界一流大学和一流学科”建设名单。
