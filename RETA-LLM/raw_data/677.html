  • 哲学院
    http://phi.ruc.edu.cn/

    中国人民大学哲学院——中国的哲学、伦理学和宗教学教学科研重镇，被誉为“中国哲学教育的工作母机”，是目前国内教授最多、规模最大、学科配备最齐全、人才培养体系最完善、具国际声誉的哲学院系。

    本科专业：
    • 哲学
      •
    • 宗教学
      •
    • 伦理学
      •
    • 政治学、经济学与哲学（PPE实验班）
      •

  • 文学院
    http://wenxueyuan.ruc.edu.cn

    中国人民大学文学院的前身是1939年由陕北公学、延安鲁迅艺术学院等合并成立的华北联合大学文艺学院，是中国新文艺孕育和发展的摇篮。

    本科专业：
    • 汉语言文学
      •
    • 汉语言
      •

  • 历史学院
    http://lsxy.ruc.edu.cn/

    中国人民大学的历史学科源自1937年成立的陕北公学，办学历史悠久，力量雄厚。历史学院由历史系、考古文博系、清史研究所和民国史研究院组成，学科门类齐全，学术成就显著，在国内外享有盛誉。

    本科专业：
    • 历史学
      •
    • 考古学
      •
    • 世界史
      •

  • 艺术学院
    http://art.ruc.edu.cn/

    中国人民大学艺术教育源自陕北公学，1999年艺术学院复建，经过23年的建设与发展，不断凝练学科建设方向，在人才培养质量、学科影响力、国际交流等多个方面得到显著提升。

    本科专业：
    • 环境设计(景观建筑方向)
      •
    • 动画(新媒体方向)
      •
    • 绘画(油画)
      •
    • 绘画(中国画)
      •
    • 美术学（艺术管理与策划方向）
      •
    • 视觉传达设计
      •
    • 声乐（美声方向）
      •
    • 钢琴、弦乐、管乐
      •
    • 多元音乐启蒙
      •
    • 古琴
      •

  • 外国语学院
    http://fl.ruc.edu.cn/

    中国人民大学外国语学院诞生于1937年创立的陕北公学以及后来的华北联合大学和华北大学的俄语系。学校的发展历史紧紧伴随着中国革命和国家建设的历史，是我国培养高素质复合型外语人才的重要基地。

    本科专业：
    • 英语
      •
    • 日语
      •
    • 俄语
      •
    • 德语
      •
    • 法语
      •
    • 西班牙语
      •

  • 国学院
    http://guoxue.ruc.edu.cn

    中国人民大学国学院成立于2005年，是新中国第一家以“国学”命名的高等教育科研机构。下设汉语古典学系和西域古典学系。
